main¶

Project main controller

class spam.controllers.project.main.Controller(*args, **kwargs)¶

REST controller for managing projects.

In addition to the standard REST verbs this controller defines the following REST-like methods:

  • archive (get_archive(), post_archive())
  • activate (get_activate(), post_activate())
  • upgrade (get_upgrade(), post_upgrade())
  • sidebar (get_sidebar())

get_all()¶

Return a full page for managing active and archived projects.

This page is linked from the admin sidebar.

Return a tabbed page for project tabs.

get_sidebar(proj)¶

Return a html fragment containing the project sidebar. This is used to dynamically reload the sidebar when the project structure changes without reloading the whole page.
